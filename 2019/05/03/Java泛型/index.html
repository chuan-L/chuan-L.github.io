1. 基本原理

泛型就是类型参数化。
Java中的泛型是伪泛型，是通过类型擦除实现的。在生成的字节码中，不存在泛型。原来的泛型会替换为Object，并加入了强制类型转换代码。C#中的泛型是真泛型，在源码和IL以及运行期的CLR中都存在。Java的泛型相当于一个语法糖。

2. 好处

使用泛型而不使用Object有其优点。

  1. 安全性

  2. 可读性
    不用在每一个地方都加上强制类型转换了。
